JNINJA: Objects

  • Everything in JavaScript is either one of the six primitive data types
    viz: strings, numbers, booleans, symbols, undefined and null.
  • Objects are key value pairs. Like a dictionary or JSON notation.
  • Object literals can be used in JavaScript to create a quick local scoped objects containing atrributes, variables and functions (methods)
    example:
                    const obj = {
                  key: 'value1', 
                  ...
                  nkey: nvalue,
                  function(){
                    //code block;
                  }};
    where the values could be any JavaScript data type and the key is called name.
  • Access Object value by dot notation ie; obj.key or obj['key']
  • The [] can be used to concatenate two or more key value pairs
  • Use an assignment statment to change the value of a key in an object
    or to add a new property. Example: 
                  obj.key = value; // if key exist in object its value will be overwritten with value,
                              //if not exist it will be created and value assigned.
  • Use delete keyword to delete a key value pair from the obj. eg. delete obj.key
  • Nested objects have a further key/value pair as the value of the initial key
  • Values in a nested object can be accessed using the dot notation like so: obj.key0.key1
  • Objects assignment to variables are done by reference and not by copy. Meaning both variables will take same value if one is changed.
  • JSON is a string representation of the object literal but with double quoted names (or keys) and functions are not permitted values. it is in essence a data storage format.
  • JSON.parse(obj); //converts a JSON data to JavaScript object.
  • JSON.stringify(obj);// converts a JavaScript object to a JSON data. Functions will be ignored or removed.
  • Math objects are built in math constants used in calulation. All math objects are named in CAPS.
  • Use objects literals to pass parameters to a function if there are many parameters that need to be passed.
  • use assigment statements to assign default values as needed.
  • Use object literals to create a sort of local scoped namespace. add values as properties of the object and functions as methods. This way variable names become scoped to the object and not global and conflicts in variable and function methods can be avoided for large code bases.
  • RegExp objects are used for matching string patterns. They are useful in find and replace type operations.
  • RegExp constructor: // variable = new RegExp('/patterns/'));
  • RegExp method:
    • test(); // checks for match
    • global(); //or /'patterns'/g returns all matches instead of the first one only.
    • ignorecase(); // or /'patterns'/i makes pattern non case sensitive
    • multiline(); // or /'pattern'/m makes pattern multiline.
    • This keyword is used to reference the object it is within.

The Document Object Model

  • The Document object Model or DOM is a representation of the elements on a web page. and JavaScript and other scripting languages can access this elements and directly manupulate them to change the display, functionality and behaviour of the page. This allows a web page to become interactive rather than static.
  • To get elements on a web page use one of these:
    1. getElementById //use #elementName to get element.
    2. getElementByClassName
    3. getElementByTagName
    4. querySelector
    5. querySelectorAll
  • Note that even blank objects within the HTML page is also an object node.
  • legacy DOM shortcuts using the dot notatation can be used to get all elements of type from the document object. Example:
    1. Document.images // gets all images on the page.
    2. Document.links //gets all links in the page.
    3. Document.anchors//gets all anchor tags in document.
    4. Etc
  • These tags return arrays sometimes but not all nodelist are arrays. Document.images; for example will return an array of nodelist. array methods can be used on it. And it has also a length property.
  • querySelector is a versatile selector for elements on a age. CSS pseudo selectors can be used inside a querySelector to target very precise elements on the page. Example: 
              const ul = document.querySelector('ul#roster');
                    ul.querySelector('li#bats')
  • JQuery is JavaScript framework built on this kind of CSS-style syntax.

Events

  • While the DOM provides feedback from a page to the scripting languages The Events is a DOM interface that provides feedback to the scripting language based on user interaction on the page.
  • Events include all forms of interaction on the page, onlick, onload, onresize, onmouseover Etc
  • AddEventListener is the prefered mthod to use rather than the older document.onclick method.
