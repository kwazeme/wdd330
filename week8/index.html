  • SVG, Scalable Vector Graphics: Can be used in much to same way as canvas to draw content but SVG objects are accesible via the DOM
    and since it is drawn by using XML SVG objects are also search engine friendly.
    SVG Resources:
    1. SVG Image editor
    2. Public Royalty free SVG cliparts
    3. Opensource JavaScript library for drawing and animating SVG
  • Drag and Drop API: used for dragging and dropping certain element in the page onto other elements and to specify what happens when this is done.
    This API is not supported in Android and IOs
    EventListeners hooks for Drag and Drop:
    1. dragable// attribute property for an item to indicate it can be dragged. It is not Boolean and must be set to true, like so, dragable="true"
    2. drop//Fires when you drop an item.
    3. dragover//Fires when item is dragged over another
    4. dragstart
