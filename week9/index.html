  • Window Object
    1. window// the window object is the global variable in a browser.
    2. alert(), confirm() and prompt()// these generate dialogs that terminate program execution and should be used with caution.
    3. document//document object exists within the window object
    4. Methods/Objects of the window and document objects:
      1. The window.navigator// object gives information about the user’s browser and operating system
      2. The window.location // object provides information about the URL of the current page.
      3. The window.history //object keeps information about the pages that have been visited in the session.
      4. The window.screen // object provides information about the user’s screen.
      5. document.write() //is an archaic method of writing text to the document and should be avoided.
      6. Use Cookies to store small pieces of information between requests using the document.cookie property.
      7. The window.setTimeout() //method can be used to invoke a function after a set amount of time. It can be canceled using the clearTimeout() method.
      8. The window.setInterval() //method can be used to repeatedly invoke a function. It can be stopped using the clearInterval() method.
      9. The window.requestAnimationFrame()// method can be used to produce smooth and optimized animation by utilizing the browser's built-in graphics capabilities. It can be canceled using the cancelAnimationFrame() method.
    5. Shims and Polyfills are used to assist with broswer compactibility with newer APIs
  • HTML5 APIs
    1. HTML5 is latest HTML version and includes several API
      1. Web storage API - allows the storage of key-value pairs as found in localStorage.
      2. Geolocation API - used to access the coordinates of the user device when permission is allowed.
      3. Web Worker API - for performing intensive computations
      4. Web Sockets API - exposes a protocol for two way communication in real time throught the browser
      5. NOtification API - used to display notifications on user device.
